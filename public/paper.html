Research Paper

3we: An Open Platform for Sim-to-Real Embodied AI Research

3we Research Team

download Download PDF open_in_new View on arXiv

Abstract

We present 3we, an open-source platform that unifies simulation and real-world deployment for embodied AI research. The platform introduces a backend abstraction layer that enables identical Python code to execute across four interchangeable environments: a zero-dependency mock simulator, Gazebo Harmonic for physics-accurate simulation, NVIDIA Isaac Sim for GPU-accelerated parallel training, and physical hardware based on Raspberry Pi 5 with ESP32-S3 controllers. Combined with reproducible open hardware costing under $500 and standardized Gymnasium environments, 3we eliminates the three primary barriers to embodied AI research: prohibitive hardware costs, sim-to-real transfer complexity, and lack of reproducible benchmarks. We demonstrate that policies trained in simulation transfer to real hardware with zero code modifications, achieving 85%+ success rates on navigation tasks. The platform includes three standardized benchmark tasks (PointNav, ObjectNav, Exploration) evaluated across 8 scenes with a public leaderboard.

Key Contributions

  1. 1.

    Backend abstraction layer enabling zero-modification sim-to-real transfer

  2. 2.

    Open hardware design (<$500) with full BOM, PCB schematics, and assembly guides

  3. 3.

    AI-First Python API reducing code complexity by 12x compared to raw ROS2

  4. 4.

    Standardized Gymnasium environments for embodied AI benchmarking

  5. 5.

    PBC-34 hot-plug payload bus for modular hardware extension
